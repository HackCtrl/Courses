
***Менеджер пакетов*** - это программа, которая управляет установкой, обновлением и удаление ПО в ОС. (автоматизирует установку)

***Пакет ПО*** - программы автоматизируют установку компонентов

***Основные функции пакетных менеджеров:*** 
- Загрузка и установка ПО 
- Обновление уже установленных пакетов до последних версий 
- Удаление ПО и связанных с ними файлов 
- Управление зависимостями: библиотеками и другими пакетами, необходимыми для работы установленного ПО
## **Траблшутинг**

#### #### **Проблема №1. Конфликты между пакетами**

Конфликты между старыми и новыми пакетами 
#### **Решение**
Удалить старую версию, а затем установить нужную 
#### **Проблема №2. Ошибка при установке**

Отсутствие доступа к репозиторию пакетов 
#### **Решение**
Проверка корректности сетевых настроек виртуальной машины и подключение сетевого адаптера.

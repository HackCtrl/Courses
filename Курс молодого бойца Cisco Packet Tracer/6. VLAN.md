***VLAN*** - Virtual Local Area Network

***Коммутатор внутри коммутатора. Преимущества:***  
- VLAN помогает структурировать сеть 
- VLAN используется для обеспечения безопасности
- VLAN используется для объединения
- VLAN уменьшает количество широковещательного трафика 

***Типы портов:*** 
- Access Port - для подключения конечных устройств
- Trunk Port - для соединения между коммутаторами

1. Создаем VLAN
2. Определяем Access порты
